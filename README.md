# AI-HTTP-Tester

AI HTTP Tester 🚀
An automated tool for security interns to turn 2 hours of manual HTTP testing into 10 minutes.

⚡ The Workflow
1. Paste a Raw HTTP request.
2. AI Finds vulnerable parameters (SQLi, XSS, Path Traversal).
3. Automate 20+ payloads with one click.
4. Proxy requests through Node.js to bypass CORS.

🛠️ Architecture

Frontend: React (State Management & UI)

Backend: Node.js + Express (CORS Proxy)

Intelligence: Groq API (Vulnerability Analysis)

🚀 Quick Start
1. Install: npm install in both /client and /server.
2. Config: Add OPENAI_API_KEY to server/.env.
3. Run Backend: cd server && node index.js.
4. Run Frontend: cd AI-HTTP-TESTER && npm run dev.
